Case Study

SoZo:
The Social NFT Marketplace

A product design case study analyzing the visual hierarchy, design system, and strategic UX decisions behind a community-driven Web3 platform.

ROLE

Product Designer

TIMELINE

3-4 Months (2023)

TOOLS

Figma, Webflow, After Effects

1. Design Objectives & Strategy

Primary Goals

  • Establish SoZo as a trustworthy, secure blockchain-based platform.
  • Simplify complex Web3 technology into an accessible interface.
  • Create a community-first experience.
  • Position the brand as innovative and modern.

Target Audience

  • Creators: Seeking community value.
  • Collectors: Interested in social validation.
  • Web3 Enthusiasts: Early adopters.
  • Newcomers: Users intimidated by crypto.

2. Visual Design System

Color Palette

The dark background with vibrant accent colors creates a premium, tech-forward aesthetic. This palette positions SoZo as innovative while maintaining sophistication—crucial for a financial product.

3. Layout & Information Architecture

The layout strategy focuses on clear spatial hierarchy, using ample negative space and consistent padding to prevent cognitive overload.

Homepage Structure

  • Hero: Centered headline, gradient background, dual CTAs.
  • Features: 2x2 Grid system presenting the four core mechanics.
  • Problem Statement: Circular visual metaphor connecting issues.

Spatial Hierarchy

  • Ample negative space between sections.
  • Consistent padding (8px/16px scale).
  • Centered alignment for focus.

4. Component Design

Interactive Buttons

Designed for high contrast and clarity.

Primary Action Secondary

Primary: Electric Blue, rounded corners, hover glow.
Secondary: Outline style, lower visual weight.

Feature Cards

Consistent dark background with subtle 1-2px borders. Uses a numbered badge system (01, 02, 03) to create visual rhythm and reading order.

5. Visual Elements & Iconography

The Infinity Logo: The gradient infinity loop (blue → purple → orange) acts as the primary brand mark and a metaphor for limitless possibilities.

Imagery: We utilize 3D abstract geometric shapes (cubes, torus, networking visuals) in warm palettes to contrast against the dark background, creating a sense of innovation.

6. UX Patterns & User Journey

Despite the blockchain foundation, we prioritized Clarity over Complexity. We utilize progressive disclosure to guide the user from the high-level value proposition down to specific mechanics.

  • Plain Language: Avoiding technical jargon in favor of accessible terms.
  • Social Proof: "Meet the Team" sections build trust.
  • Micro-interactions: Smooth hover effects on cards and icon transitions to increase engagement.

7. Design System Consistency

The site is built on a rigorous framework to ensure scalability across devices.

  • Grid: 12-column responsive grid.
  • Spacing: 8px base unit (16px, 24px, 32px increments).
  • Radius: 8-12px for medium elements, 16px for containers.
  • Transparency: Semi-transparent overlays prevent visual heaviness.

8. Analysis & Outcomes

Strengths

  • ✓ Strong Visual Identity: Instantly recognizable gradient branding.
  • ✓ Accessibility: High contrast ratios.
  • ✓ Emotional Design: Colors evoke innovation and trust.
  • ✓ Mobile Responsiveness: Excellent adaptability.

Opportunities

  • Add interactive "Expand" states to feature cards.
  • Implement scroll-triggered storytelling animations.
  • Develop a component library documentation.
  • Add video demonstrations of core features.
